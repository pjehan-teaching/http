Réponse

Présentation

Une réponse HTTP est composée en 3 parties :

Status Line
  • HTTP Version : Version du protocole utilisé
  • Status Code : Code renvoyé par le serveur (200, 404...)
  • Reason Phrase : Version texte du code (OK, Not Found...)
Request Headers
  • Server : Informations sur le serveur (Apache, Nginx...)
  • Content-Type : Type de contenu retourné (text/html, application/json...)
  • Content-Length : Taille du contenu retourné
Request Body Le contenu de la réponse
